[![Build Status](https://travis-ci.org/stevegood/toastmasters.svg?branch=master)](https://travis-ci.org/stevegood/toastmasters)

## Dependencies

- npm
- Bower
- Grunt

## Setup

Execute ```npm install``` from the project root to download all of the dependencies.
